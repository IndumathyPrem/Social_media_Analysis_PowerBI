 Social Media Analytics Dashboard (Power BI)

 Overview

This project presents an interactive **Power BI dashboard** built using a social media dataset.
The dashboard provides insights into user engagement, activity patterns.

---

Objectives

* Analyze user engagement (likes, comments)
* Identify top influencers
* Track user activity trends
* Monitor follower growth
* Detect trending hashtags

Dashboard Features

 Overview

* Total Users
* Total Posts
* Total Likes
* Total Comments

---
Top Influencers

* Displays users with highest engagement
* Based on likes + comments

---
 Engagement Analysis

* Post-level engagement (likes & comments)
* Identifies viral posts

---
 Email Domain Distribution

* Pie chart showing user distribution by email providers
* Helps understand user segmentation

---
 Tools & Technologies

* Power BI Desktop
* DAX (Data Analysis Expressions)
* MySQL (Data Source)


---

 How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Refresh data (if connected to database)
---

 Key Insights

* Certain users generate significantly higher engagement
* Posting frequency varies across days
* Engagement is unevenly distributed across posts
* Email domain distribution shows user preference trends
